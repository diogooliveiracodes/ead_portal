# Portal EAD
Protótipo de um portal EAD com Landing Page e portal do aluno.

## 🚀 Funcionalidades
* Login e Cadastro.
* Portal do aluno.
* Área de suporte.
* Consumo de cursos e aulas cadastradas via CRM.

## 💻  Pré-requisitos
* PHP 7.2.5
